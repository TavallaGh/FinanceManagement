Definition of Ready برای شروع اجرای تیم

چک‌لیست حداقلی قبل از شروع هر Task/Story

Team Readiness

Ready for Solution

  • Refienment کامل و تایید شده
  • AoC/DoD واضح و بدون تناقض
  • حدود Scope مشخص

Ready for Implementation

  • Task Spec کامل (Expectation/TDD/BDD/DoD)
  • Dependencies و Risks ثبت شده
  • Jira metadata کامل + Fix Version صحیح

Ready for Completion

  • Implementation evidence کامل
  • MR/Issue/Jira links ثبت شده
  • Validation notes و rollback notes موجود

قانون طلایی

اگر Artifact فاز قبل ناقص است، فاز بعد نباید شروع شود. این قانون هزینه بازکاری را به شدت کاهش می‌دهد و کیفیت تحویل را قابل پیش‌بینی می‌کند.
