Solution — تبدیل تحلیل به بسته اجرایی

ساخت Task Package استاندارد برای اجرا و import به Jira

01.solution

وظیفه اصلی Solution

  • تبدیل وضوح Refienment به Taskهای قابل اجرا
  • ساخت یک Task Plan تجمیعی در سطح Story
  • ساخت یک فایل Markdown مستقل برای هر Task

Paths (کلیدی)

  • docs/work-items/01.solution/linked/stories/<STORY-KEY>/solution.md
  • docs/work-items/01.solution/linked/stories/<STORY-KEY>/task-plan.md
  • docs/work-items/01.solution/linked/stories/<STORY-KEY>/tasks/<TASK-KEY>.md

حداقل محتوای هر Task

  • Description
  • Goal Of Task
  • Problem To Solve
  • AoC + Scope + Expectations
  • TDD + BDD + DoD
  • Traceability Sources

Rule مهم

اگر Taskها برای یک Story هستند، بعد از تایید Solution باید به صورت Subtask زیر همان Story در Jira ثبت شوند.
