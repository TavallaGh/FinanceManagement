ورود به Tech Stack منتخب

Backend Core

  • .NET 10 + ASP.NET Minimal API
  • CQRS with MediatR
  • Endpoint در ماژول، Host فقط Register

Frontend + Data

  • Angular SPA + BFF Architecture
  • SQL Server (OLTP)
  • Redis برای Read Caching

Security + Observability

  • SSO Service (accounting-sso) + JWT Validation
  • Notification: accounting-notification (Email/SMS/OTP)
  • Serilog -> ELK
  • Health Checks + Metrics

پیام معماری

اصل کار این است: در MVP با معماری یکپارچه سریع جلو می‌رویم، اما قواعد مرزبندی را طوری نگه می‌داریم که استخراج سرویس‌ها بعدا کم‌ریسک باشد.

پیام اجرایی

این اسلاید نقطه ورود سکشن 3 است: اول قواعد و فناوری را شفاف می‌کند، سپس در اسلایدهای بعدی دیاگرام‌ها و جداول تصمیم را جزئی می‌کند.

پل به اسلاید بعدی

در ادامه، سه دیاگرام کلیدی (Context/Container/Layered) نشان می‌دهند این تصمیم چطور در طراحی واقعی سیستم پیاده شده است.
