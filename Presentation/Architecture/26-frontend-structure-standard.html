Frontend Structure (Nx Monorepo)

Angular Root

accounting-frontend

  • apps/erp-web/ (main app)
  • libs/shared/ (UI, data-access, util)
  • libs/domains/ (domain libraries)
  • public/ (assets, tokens)
  • docs/frontend/ (documentation)

App Structure

apps/erp-web/src/app/

  • core/ (app-wide infrastructure)
  • features/ (business modules)
  • shared/ (app-scoped helpers)
  • Modern Angular (signals, inject())
  • BFF برای authentication و API

Responsibilities

  • Components: UI فقط، بدون HTTP مستقیم
  • Services: Business logic و BFF calls
  • Design System: Token-based styling
  • i18n: Flat uppercase keys
  • RTL/LTR: Logical CSS properties

این ساختار با Nx dependency boundaries، جداسازی لایه‌ها و الگوهای Modern Angular، توسعه را سریع‌تر و قابل نگهداری‌تر می‌کند.

Rule Check
