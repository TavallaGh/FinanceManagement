# MVP Now (Frontend Delivery)

## Scope Rule

Include all screens required by `docs/mvp/01..13` with working shell, auth, localization, and permissions.

## Deliverables

1. Shell and navigation
- App shell with module rail, tree menu, workspace.
- Header actions and contextual documentation access.

2. Auth
- Login modes and password recovery states.
- Error rendering and bilingual messages.

3. Access control
- Role-based and direct-permission-aware menu filtering.
- Form/action-level UI restrictions.

4. Operational setup screens
- Organization info
- Users
- Roles
- Branches
- Parties
- Cost centers
- Projects
- Detail types
- Fiscal periods
- Ledgers
- Auto numbering

## Exit Criteria

- User can login, navigate authorized menus, and configure core master data and GL setup screens.
- UI works in both `fa` (RTL) and `en` (LTR).
- MVP 01-13 pages are accessible and usable from shell navigation.
