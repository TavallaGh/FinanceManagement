# Solution Task Spec: AC-72

- Stack: Frontend / UI
- Status: draft

## 2. Description

Implement `UiSimpleListComponent` and `UiSimpleListItemComponent` as Angular standalone components in `libs/shared/ui/src/lib/components/simple-list/`. The component provides a reusable list container with a canonical item structure. Item content is projected via `ng-content`, making the component business-agnostic. All styling is token-driven, theme-aware, and RTL/LTR-safe. Export from `index.ts`. Add a Storybook page at `story-book/pages/simple-list/`. Register the route.

## 3. Goal Of Task

Provide a shared list primitive with a consistent item wrapper so feature UIs stop building one-off flat lists with inconsistent item spacing, dividers, and hover states.

## 4. What Problem This Task Should Be Done For

Feature lists are inconsistent — each feature wraps list items differently, with divergent spacing, border separators, and hover treatments. Without a shared item wrapper, global list styling changes require editing every feature's list implementation.

## 5. AoC (Acceptance of Completion)

- AOC-01: `UiSimpleListComponent` implemented as a list container with `ng-content` projection for `UiSimpleListItemComponent` children.
- AOC-02: `UiSimpleListItemComponent` implemented with: `ng-content` default slot for item content; optional `selected` input (boolean); optional `disabled` input (boolean).
- AOC-03: Item separator (border or divider) between items rendered via token-driven border color.
- AOC-04: Item hover state implemented with token-driven background color; suppressed when `disabled=true`.
- AOC-05: Selected item state rendered with token-driven background color when `selected=true`.
- AOC-06: All styling uses semantic tokens exclusively; zero hardcoded values.
- AOC-07: Components use logical CSS properties for padding and border placement (RTL/LTR safe).
- AOC-08: Components use `ChangeDetectionStrategy.OnPush` and standalone pattern.
- AOC-09: Storybook page at `story-book/pages/simple-list/` covers: basic list, list with selected item, list with disabled item, content-rich item projection, RTL layout.
- AOC-10: Both components exported from `libs/shared/ui/src/lib/components/index.ts`.
- AOC-11: Route registered in `story-book.routes.ts`; navigation entry in Storybook shell sidebar.
- AOC-12: Translation keys for Storybook demo content added to `en.json` and `fa.json`.

## 6. Scope

- In Scope:
  - `libs/shared/ui/src/lib/components/simple-list/` — `UiSimpleListComponent` and `UiSimpleListItemComponent`
  - Content projection, selected state, disabled state, hover state
  - Storybook page and route registration
- Out Of Scope:
  - Pagination (that is TBD-10)
  - Drag-and-drop reordering
  - Virtual scrolling
  - List filtering or search within the component
